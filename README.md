# Rails Products AI

ChatBot - Rag test app on RoR

## Technologies Used

- Ruby on Rails 7
- PostgreSQL
- OpenAI API Integration
- Bootstrap 5
- Stimulus.js
- Turbo

## Prerequisites

- Ruby 3.2+
- PostgreSQL
- Ruby on Rails 7.2

## Installation

1. Clone the repository

```bash
git clone git@github.com:yourusername/rails-products-ai.git
```

2. Install dependencies

```bash
bundle install

```

3. Setup database

```bash
rails db:create db:migrate
```

4. Start the server

```bash
rails s
```
